# Documentation

Technical documentation for the large-translate library.

## API Documentation

| Document | Description |
|----------|-------------|
| [On-Demand Translation](./on-demand-translation.md) | Real-time translation API with intelligent chunking and context preservation |
| [Batch Translation](./batch-translation.md) | Asynchronous batch processing for cost-effective bulk translation |

## Quick Comparison

| Feature | On-Demand | Batch |
|---------|-----------|-------|
| Response Time | Immediate | Minutes to hours |
| Cost | Standard | ~50% cheaper |
| Context Continuity | Yes | No |
| Best For | Interactive use, small files | Large documents, bulk processing |

## Getting Started

For most use cases, start with the [On-Demand Translation](./on-demand-translation.md) guide. If you're processing large volumes of content and can wait for results, see [Batch Translation](./batch-translation.md) for cost savings.
